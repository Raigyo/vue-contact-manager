## How to use?

### Localy

Clone, then:

```
npm install
npm run serve
```

**During development**

Problem with spaces using Eslint, use: `npm run lint -- --fix`

Also put your pages in *LF* (=Line Feed) instead of *CR+LF* (=Carriage Return + Line Feed)

**Compiles and minifies for production**

`npm run build`

**Useful snipets**

`vbase`

`vimport`

`vmethod`

`vdata`
